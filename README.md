# README – Herança em Java (Funcionários)

## Objetivo
Praticar herança em Java utilizando extends, super, atributos protected e polimorfismo.

## Estrutura do Projeto
- Funcionario (classe abstrata)
- FuncionarioCLT
- FuncionarioPJ
- FuncionarioTest

## Conceitos Aplicados
- Herança com extends
- Chamada de construtor da superclasse com super()
- Uso de atributos protected
- Sobrescrita de métodos com @Override
- Polimorfismo com ArrayList<Funcionario>
- Classe abstrata (abstract)

## Regras de Negócio
- Funcionário CLT: desconto de 10% sobre o salário base
- Funcionário PJ: acréscimo de 15% sobre o salário base
